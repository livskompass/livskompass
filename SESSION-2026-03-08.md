## How to Test Inline Editing
1. Log into admin at :3001
2. Copy token: on :3001 console run `copy(localStorage.getItem('admin_token'))`
3. On :3000 console: `localStorage.setItem('admin_token', 'PASTE_TOKEN')`
4. Reload :3000 — edit toolbar appears, all text is editable on hover/click
